# Watermark Pro (Vodoznak)

**Author:** František Čanda

## O aplikaci / About
Tato aplikace slouží k jednoduchému a rychlému vkládání vodoznaků do fotografií. Je navržena pro maximální kvalitu výstupu a snadné použití přímo v prohlížeči.

### Hlavní funkce
-   **Vlastní vodoznak**: Nahrajte libovolný obrázek (PNG/SVG) jako vodoznak.
-   **Arandur Mód**: Speciální režim pro rychlou úpravu s přednastaveným logem Arandur (světlá/tmavá varianta).
-   **Plná kvalita**: Exportuje fotografie v původním rozlišení.
-   **Bezpečí**: Vše probíhá ve vašem prohlížeči, fotky se nikam neodesílají.

## Jak spustit / How to run

1.  Nainstalujte závislosti:
    ```bash
    npm install
    ```
2.  Spusťte vývojový server:
    ```bash
    npm run dev
    ```

## Technologie
-   React + Vite
-   Canvas API
-   Glassmorphism UI Design
